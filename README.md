# Tikona Research OS

Tikona Research OS is an internal equity research platform centered on the
`/admin/pipeline` workflow. The active stack is a Vite/React frontend, Supabase
for data and storage, a Python financial-model service, and a Python PPTX
generation service.

Primary reference docs:

- `CLAUDE.md` for repo architecture and workflow notes
- `docs/PPT_GENERATOR_WORKFLOW.md` for the current PPTX-first report pipeline
- `scripts/FINANCIAL_MODEL_DOCS.md` for the financial-model service
